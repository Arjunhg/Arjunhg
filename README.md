### 🛩️ [Hyperplane](https://github.com/Arjunhg/hyperplane)
> Real-time aircraft trajectory prediction system

- **Predictive Intelligence:** Trained a self-supervised LSTM + Autoencoder stack to "learn" flight patterns without labeled datasets.
- **Geometric Precision:** Fused 6D Kalman filters with hyperbolic intersection math to kill positional jitter in 3D space.
- **High-Stakes Streaming:** Architected for sub-second recovery; because in flight tracking, a 5-second lag is a lifetime.

`Python` `Go` `PyTorch` `FastAPI` `AWS EC2` `Hedera`

</td>

### 🤝 [PeerPath](https://github.com/Arjunhg/peerpath)
> AI-powered peer mentorship matching platform

- **Smart Discovery:** Built an LLM-engine that looks past keywords to match mentors based on actual skill compatibility.
- **Growth Engineering:** Implemented viral loops and AI-generated summaries to turn one-off chats into recurring mentorships.
- **Data-Led UX:** Designed a custom event funnel to pinpoint exactly where users drop off during the matching process.

`Next.js` `Node.js` `PostgreSQL` `OpenAI`

</td>
